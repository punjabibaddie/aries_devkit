ASSIGNMENT WEEK 1

This project is just an assignment given by ARIES to the new members to learn about bash and git.
The first script that is setup_check.sh can be run just by going to the directory ~/aries_devkit/scripts and running the command /.setup_check.sh.
to run the second script that is git_summary.sh you will have to put an extra argument that is a path to the git repository of which the summary will be displayed. To run it, again go to the path ~/aries_devkit/scripts and run the command /.git_summary.sh REPO_PATH.
